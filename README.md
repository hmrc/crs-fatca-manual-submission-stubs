
# crs-fatca-manual-submission-stubs

This is a Stub service for crs fatca manual journey

### Running the stub

This microservice is part of Service Manager: CRS_FATCA_ALL
or
Can be run individually through Service Manager: CRS_FATCA_MANUAL_SUBMISSION_STUBS
Port: 10041

Link: http://localhost:10041


### Endpoints


| Endpoint                      | Supported methods | Description                                         |
|-------------------------------|-------------------|-----------------------------------------------------|
| /dac6/getlistofsubmissions/v1 | POST              | Returns list of submission history and require FIID |
| /dac6/submitVoidRequest/v1    | POST              | Returns Created response code                       |
